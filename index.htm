A portfolio navbar with skillbars

By: Martin emanuelsson

Om pluginet

Trender kommer och trender går. Men en jQuery meny är allt du får. En trend som är populär just nu är snygga menyer som beter sig normalt tills vi rullar eller klickar på något på sajten. Snygga och funktionella navigeringsmetoder har verkligen vuxit i popularitet dom senaste åren och gör sidan mer dynamisk och populär bland besökarna.

I den här guiden kommer vi att skapa en meny som man klickar fram via en funktion i jQuery, men så att det inte stör innehållet eller tar upp för mycket plats.

Till min hjälp använder jag mig utav Twitter Bootstrap och Font Awesome som du bör installera för att det ska fungera korrekt.

På toppen av den här sidan finner du menyn så du kan du få ett exempel på hur det kan komma och se ut när vi är klara.

Om du inte vill hänga med i guiden kan du ladda ner filen direkt nedan.

CSS

Vår CSS används för att styla olika tillstånd och att övergången mellan de båda staterna. Till att börja med, låt oss lägga till några enkla stilar som förbättrar utseendet för knappen. Jag använder mig utav "transitions" för att få en mjuk övergång.

.box { opacity: 0; margin-right: 2px; margin-bottom: 6px; background-color: rgba(0, 0, 0, 0.2); vertical-align: bottom; padding-top: 16px; margin-bottom: 2px; text-align: center; display: inline-block; width: 84px; height: 80px; border-bottom: 1px solid transparent; } .box { color: #FFF; font-family: 'Montserrat', sans-serif; text-shadow: 0 -1px 0 rgba(0, 0, 0, .25); font-size: 10px; text-transform: uppercase; letter-spacing: 0.6px; text-align: center; } .box i { font-size: 2.2em; } .box:hover { background-color: rgba(0, 0, 0, 0.3); border-color: #32b3ea; } .box { -webkit-transition: 0.2s ease-in-out; -moz-transition: 0.2s ease-in-out; -o-transition: 0.2s ease-in-out; transition: 0.2s ease-in-out; } .box:hover i { text-shadow: #CCC 0px 0px 20px; -webkit-transition: 0.2s ease-in-out; -moz-transition: 0.2s ease-in-out; -o-transition: 0.2s ease-in-out; transition: 0.2s ease-in-out; } .box i { -webkit-transition: 0.8s ease-in-out; -moz-transition: 0.8s ease-in-out; -o-transition: 0.8s ease-in-out; transition: 0.8s ease-in-out; color: #CCC; } .box:hover i { color: #FFF; }

jQuery

CSS övergångar är det bästa sättet att hantera animeringen delen av vår meny. Allt vi använder jQuery för är att animera in och ut menyn via en klickfunktion. I menyn överst på sidan använder jag även en textremsa som visar en textrad som förtydligar vilken knapp man för musen över. Men det har vi inte med i exemplet nedan men följer med när man laddar ner filen.

/* Animate */ // .delay(100) = A delay sequence before the animation starts in .ms // .animate = How fast the animation goes /* Show menu */ $(document).ready(function(event) { $(".menu").stop().click(function(event) { $(".box1").stop().delay(100).animate({ opacity: '1' }, 100); $(".box2").delay(200).animate({ opacity: '1' }, 200); $(".box3").delay(300).animate({ opacity: '1', }, 300); $(".box4").delay(400).animate({ opacity: '1' }, 400); $(".box5").delay(500).animate({ opacity: '1' }, 500); $(".box6").delay(600).animate({ opacity: '1' }, 600); $(".box7").delay(700).animate({ opacity: '1' }, 700); }); console.log('Menu is ready.'); }); $(document).ready(function() { $(".menu").click(function() { $(".boxes").stop().toggle(200); }); }); /* Hide*/ $(document).ready(function() { $(".menu-hide").click(function() { $(".boxes").stop().toggle(800); }); }); $(document).ready(function() { $(".menu-hide").stop().click(function() { $(".box1").stop().animate({ opacity: '0' }, 800); $(".box2").stop().animate({ opacity: '0' }, 700); $(".box3").stop().animate({ opacity: '0' }, 600); $(".box4").stop().animate({ opacity: '0' }, 500); $(".box5").stop().animate({ opacity: '0' }, 400); $(".box6").stop().animate({ opacity: '0' }, 300); $(".box7").stop().animate({ opacity: '0' }, 200); }); }); $(".menu-hide").click(function() { $(".menu-hide").hide(); $(".menu").stop().show(); $(".menu-hide").stop(); }); $(".menu").click(function() { $(".menu").hide(); $(".menu-hide").show(); $(".menu-hide").stop(); });
